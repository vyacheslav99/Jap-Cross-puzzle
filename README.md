# Jap Cross-puzzle
Японский кроссворд. Судоку.

Версия 4.0.0.0

Сборник игр включает в себя 3 игры: Японский кроссворд, Цветной японский кроссворд и Судоку.
Сборник включает в себя небольшую базу кроссвордов (как цветных, так и черно-белых).

Кроме игр в сборник входят редакторы кроссвордов и судоку, так что вы можете создавать их
сами, как из уже существующих, так и с нуля. Редактор кроссвордов также позволяет импортировать
кроссворды из картинок (форматов BMP, JPEG, PNG, ICO, GIF) или судоку из текстовых файлов, содержащих
последовательность цифр. При импорте картинок, если размер изображения по любому из измерений 
не превышает заданного в панельке инструментов, берется масштаб 1:1, иначе изображение пропорционально 
уменьшается до наибольшего размера из указанного в панельке. Т.е. в первом случае, если у вас картинка 
размером 64х64 пикселя, то такой же и будет кроссворд - 64х64 клеточки. Можно использовать картинки 
кроссвордов с различных сайтов, посвященных этой теме. Для получения нормального кроссворда импортом 
картинки вам придеться поэкспериментировать с настройками анализатора изображений.
